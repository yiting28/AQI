從網路上抓取空氣品質指標 (AQI) 的JSON資料，並使用RecyclerView顯示每項資料，點選後可以查看詳細資料。

功能
抓取網路上的AQI資料
使用RecyclerView顯示AQI數據
點選每項AQI資料查看詳細信息
